Librarian Query

Description

Here you can add a new Librarian to the Library records. You will
need to insert the librarian's ID, name, office address
and then click submit to update the Librarian records.

Description

Here you can delete a Librarian from the Library records.
You will need to insert a Librarian ID for which you
want to delete. Warning, deleting a Librarian is a permanent
action and can not be undone.
